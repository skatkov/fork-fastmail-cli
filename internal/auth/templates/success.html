Environment Setup

Add this to your shell profile to set your default Fastmail account:

~/.zshrc or ~/.bashrc
# Fastmail CLI default account
export FASTMAIL_ACCOUNT="{{.Email}}"

This is optional. You can also use --account flag to specify the account per command.

Quick Start
List recent emails
fastmail email list --limit 10
View your masked emails
fastmail masked list
Create a new alias
fastmail masked create example.com
